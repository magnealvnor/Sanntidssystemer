Web Status Console

A page containing information about the current system version and network information. Here you can see DHCP clients connected to the network and WLAN information if avaliable. Default login is root/root or admin/admin.

AVR32 wiki documentation

A wiki snapshot of the official AVR32 documentation available from atmel.com. The wiki contains information about all AVR32 kits from Atmel.

Samba Web Administration Tool (SWAT)

SWAT can be used to configure the network file sharing on this system. By default a netdisk folder is shared.

GPIO demo

Click the buttons to the right. Two demos showing how to toggle a LED and changing LED trigger mode on the development board. Result can be observed by a blinking LED. Source code for Toggle LED and Change LED trigger. Wireing help for GPIO demo.
